Simple, lightweight plugin to embed any video on website

== Description ==

Phando Video plugin lets you embed any video on post ,pages or widgets.
It embeds the video using video link hosted anywhere.

The video runs on phando video player(phando.com),facilitating cross browser compatibility.

Player supports video sharing through Facebook, Twitter, Pinterest, Goolge+ & LinkedIn

Shortcode for embedding the video [phando src=""]

= Premium Version Features =

* play ads

== Installation ==

Upload Phando Video Plugin, activate it.

You are good to go
